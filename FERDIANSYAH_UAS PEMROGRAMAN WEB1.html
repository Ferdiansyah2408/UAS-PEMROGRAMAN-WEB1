JAWABAN NO 1
Website adalah kumpulan halaman web yang saling terhubung dan dapat diakses melalui internet. Halaman web tersebut terdiri dari teks, gambar, video, dan berbagai elemen interaktif lainnya. Secara umum, cara kerja website adalah sebagai berikut :
  1. Pengguna membuka browser dan mengetikkan alamat URL website yang ingin diakses.
  2. Browser kemudian mengirimkan permintaan HTTP ke web server yang hosting website tersebut.
  3. Web server menerima permintaan HTTP dan mengirimkan respon HTTP ke browser.
  4. Browser menerima respon HTTP dan menampilkan halaman web kepada pengguna.

Web server adalah komputer yang menjalankan perangkat lunak yang berfungsi untuk menyimpan dan mengirimkan halaman web. Domain adalah nama unik yang digunakan untuk mengidentifikasi sebuah website di internet. Hosting adalah layanan yang menyediakan ruang penyimpanan dan bandwith untuk website. Hubungan antara web server, domain, dan hosting adalah sebagai berikut :
  • Web server dan domain adalah dua hal yang berbeda, tetapi keduanya saling terkait. Domain adalah nama yang digunakan untuk mengidentifikasi web server.
  • Hosting adalah layanan yang menyediakan ruang penyimpanan dan bandwidth untuk website. Web server dan hosting saling terkait karena web server membutuhkan ruang penyimpanan dan bandwidth untuk menyimpan dan mengirimkan halaman web.
